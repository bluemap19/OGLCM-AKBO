## 📖 1. 项目概述

### 1.1 研究背景

页岩储层具有强烈的非均质性和快速的岩相变化特征，传统的测井解释方法难以实现微尺度的岩相精细表征。钻孔电成像测井提供了厘米级分辨率的地层电阻率分布图像，但如何从这些图像中自动提取岩相信息仍是一个挑战。

### 1.2 研究目标

开发一种**无监督学习**工作流程，实现：
1. 从电成像测井数据中自动提取纹理特征
2. 无需标签数据自动识别最优聚类数
3. 实现页岩微相的自动 delineation（ delineation = 圈定/表征）

### 1.3 方法名称解析

| 缩写 | 全称 | 含义 |
|------|------|------|
| **OGLCM** | Orientational Gray-Level Co-occurrence Matrix | 定向灰度共生矩阵 |
| **AKBO** | Auto-Kmeans with Bayesian Optimization | 贝叶斯优化自适配 K-means |

## 🔬 2. 算法原理

### 2.1 OGLCM - 定向灰度共生矩阵

#### 2.1.1 GLCM 基础理论

**灰度共生矩阵 (GLCM)** 是一种二阶统计纹理分析方法，通过量化图像中像素对的灰度级共现关系来描述纹理特征。

**数学定义：**

对于图像 I，给定像素对的空间关系 (d, θ)，GLCM 定义为：

```
GLCM(i, j | d, θ) = #{(p₁, p₂) | I(p₁)=i, I(p₂)=j, dist(p₁,p₂)=d, angle(p₁,p₂)=θ}
```

其中：
- `i, j`：灰度级 (通常量化为 16/32/64 级)
- `d`：像素间距离 (正整数，通常 d=1,2,3)
- `θ`：方向角度 (0°, 45°, 90°, 135°)

**归一化：**

```
P(i,j) = GLCM(i,j) / k
```

其中 k 为最大灰度级，归一化后 P(i,j) 表示概率分布。

#### 2.1.2 7 个核心纹理特征

| 特征名 | 符号 | 公式 | 物理意义 | 取值范围 |
|--------|------|------|----------|----------|
| **Energy** | ENG | Σ P(i,j)² | 纹理均匀性/能量集中程度 | [0, 1] |
| **Contrast** | CON | Σ(i-j)²·P(i,j) | 纹理清晰度/局部灰度变化 | [0, ∞) |
| **Entropy** | ENT | -Σ P(i,j)·log(P(i,j)) | 信息量/随机性度量 | [0, ∞) |
| **ASM** | ASM | Σ P(i,j)² | 角二阶矩，均匀性 (与 ENG 相同) | [0, 1] |
| **Correlation** | COR | Σ(i·j·P(i,j) - μ²)/σ² | 灰度线性相关性 | [-1, 1] |
| **Homogeneity** | HOM | Σ P(i,j)/(1+|i-j|) | 纹理平滑度/局部均匀性 | [0, 1] |
| **Dissimilarity** | DIS | Σ |i-j|·P(i,j) | 局部灰度差异强度 | [0, ∞) |

#### 2.1.3 定向特征提取策略

**"Orientational"的含义：**

本研究的 OGLCM 方法强调**多方向**和**多性质**的纹理特征提取：

1. **方向维度：**
   - 0° (水平方向) - 反映水平层理特征
   - 90° (垂直方向) - 反映垂直裂缝/韵律
   - **方向差异 (X-Y)** - 反映各向异性

2. **统计维度：**
   - **MEAN** - 多方向平均值，反映整体纹理
   - **SUB** - 子区域统计，反映局部变化
   - **X** - X 方向 (水平) 特征
   - **Y** - Y 方向 (垂直) 特征

3. **图像类型：**
   - **DYNA (Dynamic)** - 动态成像，反映相对变化
   - **STAT (Static)** - 静态成像，反映绝对电阻率

**特征命名约定：**

```
{特征类型}_{统计方式}_{图像类型}

示例：
- CON_MEAN_DYNA  = 对比度_平均值_动态图像
- ENT_X_DYNA     = 熵_X 方向_动态图像
- HOM_SUB_STAT   = 同质性_子区域_静态图像
```

**特征总数：** 7 特征 × 4 统计方式 × 1 图像类型 = **28 个纹理特征**

### 2.2 AKBO - 贝叶斯优化自适配 K-means

#### 2.2.1 传统 K-means 的局限性

| 问题 | 描述 | 影响 |
|------|------|------|
| **K 值选择** | 需要预先指定聚类数 | 依赖经验/试错 |
| **初始质心** | 随机初始化 | 易陷入局部最优 |
| **评估主观** | 肘部法则/轮廓系数需人工判断 | 结果不稳定 |

#### 2.2.2 AKBO 核心创新

**将 K 值选择和质心初始化公式化为黑盒优化问题：**

```
max K∈[Kmin,Kmax] UIndex(K)
```

其中 UIndex 是复合聚类质量指标。

#### 2.2.3 复合聚类质量指标 UIndex

**UIndex 综合三个经典指标：**

```
UIndex = SI × (1/DBI) × DVI
```

**1. 轮廓系数 (Silhouette Index, SI)**

衡量样本与其所属簇的紧密度和与其他簇的分离度：

```
s(i) = (b(i) - a(i)) / max(a(i), b(i))

SI = (1/N) × Σ s(i)
```

其中：
- `a(i)` = 样本 i 到同簇其他样本的平均距离
- `b(i)` = 样本 i 到最近邻簇的平均距离
- **SI ∈ [-1, 1]**，越大越好

**2. Davies-Bouldin Index (DBI)**

衡量簇内距离与簇间距离的比值：

```
DBI = (1/K) × Σ max(j≠i) [(d_i + d_j) / d(c_i, c_j)]
```

其中：
- `d_i` = 簇 i 内样本到质心的平均距离
- `d(c_i, c_j)` = 簇 i 与簇 j 质心间的距离
- **DBI ∈ [0, ∞)**，越小越好

**3. Dunn Index (DVI)**

衡量最小簇间距离与最大簇内距离的比值：

```
DVI = min(i≠j) d(c_i, c_j) / max(k) diam(k)
```

其中：
- `diam(k)` = 簇 k 的最大直径
- **DVI ∈ [0, ∞)**，越大越好

**UIndex 的优势：**
- SI 关注个体样本的聚类质量
- DBI 关注簇间分离度 (对噪声敏感)
- DVI 关注簇的紧凑性 (适用于任意形状)
- 三者互补，综合评估更全面

#### 2.2.4 贝叶斯优化框架

**高斯过程 (Gaussian Process) 代理模型：**

```
UIndex ~ GP(μ(K), k(K, K'))
```

**协方差函数 (RBF 核)：**

```
k(K, K') = σ² · exp(-||K - K'||² / (2l²))
```

**采集函数 (Expected Improvement, EI)：**

```
EI(K) = E[max(0, UIndex(K) - UIndex_best)]
```

平衡**探索**(exploration)与**利用**(exploitation)。

#### 2.2.5 AKBO 算法流程

```
算法：AKBO 无监督聚类
输入：数据 X, K 搜索范围 [Kmin, Kmax], 最大迭代次数 T
输出：最优聚类数 K*, 聚类标签 L, 质心 C

Step 1: 初始化
  1.1 随机选择 n_init 个 K 值 ∈ [Kmin, Kmax]
  1.2 对每个 K：
      - 执行 K-means++ 初始化
      - 运行 K-means 至收敛
      - 计算 UIndex(K)
  1.3 构建初始数据集 D = {(K_i, UIndex_i)}
  1.4 初始化高斯过程模型 GP

Step 2: 贝叶斯优化迭代 (t = 1 to T)
  2.1 基于 D 训练 GP 模型
  2.2 通过最大化采集函数选择下一个 K_t：
      K_t = argmax_K EI(K; GP)
  2.3 执行 K-means++ 聚类 (K=K_t)
  2.4 计算 UIndex(K_t)
  2.5 更新数据集 D = D ∪ {(K_t, UIndex_t)}
  2.6 检查收敛：
      - 若 UIndex 连续 n_patience 次改进 < ε，则终止

Step 3: 输出最优结果
  3.1 K* = argmax_{K∈D} UIndex(K)
  3.2 返回对应聚类标签 L 和质心 C
```

#### 2.2.6 K-means++ 初始化

相比随机初始化，K-means++ 保证初始质心分散：

```
1. 随机选择第一个质心 μ₁
2. 对每个样本 x_i，计算 D(x_i) = min_j ||x_i - μ_j||²
3. 以概率 D(x_i)/ΣD(x) 选择下一个质心
4. 重复直到选择 K 个质心
```
