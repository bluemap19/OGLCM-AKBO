## 项目概述

**OGLCM-AKBO** = **定向灰度共生矩阵 (Oriented Gray-Level Co-occurrence Matrix)** + **贝叶斯优化自适配 K-means (Auto-Kmeans with Bayesian Optimization)**

**研究目标：** 基于测井图像纹理特征，使用无监督聚类算法自动识别页岩微相，实现岩性自动表征。

**核心创新：**
1. 使用 OGLCM 提取 28 个纹理特征
2. 基于随机森林特征重要性分析，选择 4 个最优特征
3. 使用贝叶斯优化自动确定最优 K 值
4. 提出 UIndex 复合指标评估聚类质量

## 算法原理

### 1. K-means 聚类

**目标函数：**
```
J = Σ(k=1 to K) Σ(x∈C_k) ||x - μ_k||²
```

其中：
- K: 聚类数
- C_k: 第 k 个聚类的样本集合
- μ_k: 第 k 个聚类的质心

**优化方法：** K-means++ 初始化 + Lloyd 算法迭代

---

### 2. 贝叶斯优化

**优化目标：**
```
max K∈[2,10] UIndex(K)
```

**流程：**
```
初始化 (5 个点) → 高斯过程建模 → EI 采集函数 → 采样新 K 值 → 迭代 (30 次)
```

**高斯过程代理模型：**
- 核函数：RBF + ConstantKernel
- 优化器：L-BFGS-B

**EI 采集函数：**
```
EI(x) = (μ - f_best) × Φ(Z) + σ × φ(Z)
```
其中：
- μ: GP 预测均值
- σ: GP 预测标准差
- Z = (μ - f_best) / σ
- Φ: 标准正态分布 CDF
- φ: 标准正态分布 PDF

---

### 3. GMM 后验概率

**高斯混合模型：**
```
P(x|θ) = Σ(k=1 to K) π_k × N(x|μ_k, Σ_k)
```

**后验概率：**
```
P(k|x) = π_k × N(x|μ_k, Σ_k) / P(x|θ)
```

用于计算每个样本属于各聚类的概率。

## 核心公式

### UIndex 复合聚类质量指标 ⭐

**Doctor 提出的 UIndex 公式：**

```
UIndex(K) = 1 / (0.1/SI(K) + DBI(K)/1.0 + 0.01/DVI(K))
```

**其中：**
- **SI (Silhouette Index)**: 轮廓系数，衡量样本聚类质量
  - 取值范围：[-1, 1]
  - **SI 越大，聚类质量越好**
  
- **DBI (Davies-Bouldin Index)**: 簇间分离度
  - 取值范围：[0, ∞)
  - **DBI 越小，聚类质量越好**
  
- **DVI (Dunn Index)**: 簇紧凑性
  - 取值范围：[0, ∞)
  - **DVI 越大，聚类质量越好**

**公式设计原理：**

| 指标 | 特性 | 在公式中的作用 | 效果 |
|------|------|---------------|------|
| **SI** | 越大越好 | 0.1/SI | SI 越大 → 0.1/SI 越小 → 分母越小 → UIndex 越大 ✅ |
| **DBI** | 越小越好 | DBI/1.0 | DBI 越小 → DBI/1.0 越小 → 分母越小 → UIndex 越大 ✅ |
| **DVI** | 越大越好 | 0.01/DVI | DVI 越大 → 0.01/DVI 越小 → 分母越小 → UIndex 越大 ✅ |

**权重设计：**
- SI 权重：0.1（主要贡献）
- DBI 权重：1.0（基准）
- DVI 权重：0.01（微调）

**质量评价标准：**
- UIndex > 0.5: 优秀
- UIndex > 0.2: 良好
- UIndex < 0.2: 需改进

---

### SI (轮廓系数)

**定义：**
```
SI(i) = (b(i) - a(i)) / max{a(i), b(i)}
```

其中：
- a(i): 样本 i 到同簇其他样本的平均距离（凝聚度）
- b(i): 样本 i 到最近其他簇样本的平均距离（分离度）

**整体 SI：**
```
SI = mean(SI(i)) for all i
```

---

### DBI (Davies-Bouldin Index)

**定义：**
```
DBI = (1/K) × Σ(k=1 to K) max(j≠k) [(σ_k + σ_j) / d(c_k, c_j)]
```

其中：
- σ_k: 第 k 簇的平均距离
- d(c_k, c_j): 簇 k 和簇 j 的质心距离

---

### DVI (Dunn Index)

**定义：**
```
DVI = min{d(C_i, C_j)} / max{diam(C_k)}
```

其中：
- d(C_i, C_j): 簇 i 和簇 j 之间的最小距离
- diam(C_k): 簇 k 的直径（最大距离）

**近似计算（O(N) 复杂度）：**
```
diam(C_k) ≈ 2 × max{||x - c_k||} for x ∈ C_k
```

### UIndex 计算实现

**文件：** `src/akbo_clustering.py`

**函数：** `compute_uindex(X, labels)`

```python
def compute_uindex(X, labels):
    """
    计算 UIndex (使用 Doctor 提出的公式)
    
    UIndex(K) = 1 / (0.1/SI(K) + DBI(K)/1.0 + 0.01/DVI(K))
    """
    # 1. 计算 SI
    si = silhouette_score(X, labels, metric='euclidean')
    
    # 2. 计算 DBI
    dbi = davies_bouldin_score(X, labels)
    
    # 3. 计算 DVI
    dvi = compute_dunn_index(X, labels, use_approximation=True)
    
    # 4. 处理边界情况
    if np.isinf(dvi) or dvi == 0:
        dvi = 10.0
    if np.isinf(dbi):
        dbi = 0.001
    if si <= 0:
        si = 0.001
    
    # 5. 计算 UIndex
    uindex = 1.0 / (0.1/si + dbi/1.0 + 0.01/dvi)
    
    return uindex, {'si': si, 'dbi': dbi, 'dvi': dvi, 'uindex': uindex}
```

### 边界情况处理

1. **SI ≤ 0:** 设置为 0.001（避免除以 0）
2. **DBI = ∞:** 设置为 0.001（避免除以 0）
3. **DVI = 0 或 ∞:** 设置为 10.0（合理近似）
